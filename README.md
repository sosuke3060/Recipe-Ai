# Recipe-Ai

# 献立提案AIアプリ

## プロジェクト概要

本プロジェクトは、ユーザーの運動量、手持ちの食材、調理可能時間を考慮し、最適な献立を提案するアプリです。

## 主な機能

- ユーザーの運動量、手持ちの食材、調理可能時間を考慮した最適な献立を提案
- 入力された食材を基にしたレシピ提案
- 指定された調理可能時間内で完成できるレシピのみを提案
- AIとの会話による献立の深掘りが可能

## 使用技術

- 言語モデル: OpenAI (gpt-4o)
- フロントエンド: Streamlit
- バックエンド処理: LangChain
